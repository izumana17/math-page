Event bubble directs an event to its taget.
1. When an element is clicked an event is directed to the element.
2. If an event handler is set for the element, the event handler isstrats to work.
3. And after finishing event for the certain element the event starts to go upwards to the parent.
4. It continues to go up by continuously finishing current event, until it reaches the top the document element.
